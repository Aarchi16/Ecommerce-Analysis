# E-Commerce Sales & Customer Analytics

## Introduction
This project focuses on analyzing e-commerce sales data to understand sales trends, customer behavior, and platform-wise performance. The analysis was performed using Python and Excel.

## Objective
The main objective of this project is to analyze sales data and generate insights such as monthly sales trends, top customers, and profit comparison across different platforms.

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- MS Excel
- CSV Dataset

## Dataset Files
The project uses the following datasets:
- sales.csv
- monthly.csv
- international.csv
- profit.csv

## Analysis Performed
The following analysis was performed in this project:
- Sales by Platform Analysis
- Monthly Sales Trend Analysis
- Top 10 Customers Analysis
- Profit Comparison by Platform

## Project Output
The project generates the following graphs:
- Sales by Platform
- Monthly Sales Trend
- Top 10 Customers by Sales
- Profit Comparison by Platform

## How to Run the Project
1. Install Python
2. Install required libraries using:
   pip install pandas matplotlib openpyxl
3. Run the Python file:
   python analysis.py

## Conclusion
This project helps in understanding sales performance, customer contribution, and profit comparison. Data visualization helps businesses make better decisions based on sales trends.
